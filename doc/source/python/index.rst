
python知识
================

   Python是一种多范式编程语言。它不强制程序员采用特定的编程风格，而是允许几种风格：完全支持面向对象编程和结构化编程，
   并且有许多语言功能支持功能编程和面向方面编程（包括元编程和“通过魔法”方法）。

   Python的设计是高度可扩展的，而不是要求将所有需要的功能都内置到语言的核心中。新的内置模块可以很容易地用C、C++或Cython编写
   。python还可以用作现有模块和需要可编程接口的应用程序的扩展语言。
