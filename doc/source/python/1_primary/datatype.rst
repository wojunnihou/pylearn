列表
------------

    pop也是对列表的下标进行检索删除，返回删除值：

        >>> L=[7,8,9,10]
        >>> L.pop(2)
        9
        >>> L
        [7, 8, 10]

    remove(item)方法是直接对可变序中的元素进行检索删除，返回的是删除后的列表,不返回删除值（返回None）

        >>> L=[7,8,9,10]
        >>> L.remove(7)
        >>> L
        [8, 9, 10]
