   从而python作为一种简单而高效到程序语言脱颖而出,并渗入到诸多领域。

   python特点：
       1. 易于学习

       #. 易于移植
       #. 易于维护
       #. 易于扩展
       #. 标准库丰富
       #. 开源资源丰富
       #. 可嵌入
       #. 图形化界面
       #. 开发高效
